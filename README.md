# Fut-Insight

Sistema para análise, organização e acompanhamento de partidas, jogadores e estatísticas no estilo Ultimate Team / Weekend League.

## Sobre o projeto

O **Fut-Insight** é uma aplicação desenvolvida para ajudar no controle de desempenho de partidas, elenco e campanhas, trazendo uma visão mais organizada dos dados do jogador.

Com ele, é possível registrar partidas, acompanhar estatísticas, visualizar históricos e gerenciar jogadores de forma mais prática.

## Funcionalidades

- Cadastro e gerenciamento de jogadores
- Registro de partidas
- Histórico de confrontos
- Estatísticas gerais do elenco
- Dashboard com visão resumida de desempenho
- Organização de campanhas
- Preenchimento automático de dados de jogadores
- Estrutura preparada para expansão

## Tecnologias utilizadas

- React
- Vite
- JavaScript
- Electron
- CSS

## Como executar o projeto

1. Clone o repositório
git clone https://github.com/LHRibeiro10/Fut-Insight-.git
2. Acesse a pasta
cd Fut-Insight-
3. Instale as dependências
npm install
4. Rode o projeto
npm run dev
Build do projeto

Para gerar a versão de produção:

npm run build
Objetivo

O projeto foi criado com o objetivo de oferecer uma ferramenta prática para acompanhamento de desempenho no EA FC, permitindo uma análise mais detalhada da evolução do jogador, campanhas e elenco.
